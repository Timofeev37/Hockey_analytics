README - Аналитическое веб-приложение KHL Analytics

Это приложение — интерактивный веб-инструмент для визуализации
статистики КХЛ:
командных рейтингов, рейтингов игроков и карт бросков/голов/xG.
Написано на Python, использует Flask на бэкенде и HTML на
фронтенде.

Возможности приложения

1. Командная аналитика — best_teams.py

Страница: /teams

-   Выбор фильтра по любому статистическому столбцу:
        ЗШ - заброшенные шайбы
        ПШ - пропущенные шайбы
        РШ - разница шайб
        xG - ожидаемые голы
        xGA - ожидаемые пропущенные голы
        xGD - ожидаемая разница шайб
        Net - расчетный общий рейтинг команды
        Off - расчетный атакующий рейтинг команды
        Def - расчетный защитный рейтинг команды
        Плей-офф - вероятность выхода в плей-офф
        Команда
-   Выбор порядка сортировки
-   Генерация графика рейтингов команд
-   Перерисовка графика при смене фильтра
-   Сохранение графики в /static/images/

2. Рейтинг лучших и худших вратарей — best_players.py

Страница: /players

-   Фильтрация по спасениям выше ожидаемого (GSAx)
-   Сортировка
-   Визуализация топ-10 и боттом-10 игроков
-   PNG‑график

3. Карты бросков — shots_map.py

Страница: /shots

Поддерживаемые фильтры:

-   season (сезон)
-   stage (этап чемпионата)
-   pos (позиция игрока)
-   format (формат игры)
-   team (команда)
-   player (игрок)
-   graphic (Тепловая / Точечная)
-   map_type (динамический выбор)

Функции:

-   generate_heat_map (создание тепловой карты)
-   generate_heat_map_points (создание карты бросков с ожиданием xG)

Карта сохраняется в /static/images/heat_map.png
